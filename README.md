## ⚙️ Como executar? 
1. Certifique-se de ter o Java instalado:
    ```bash
    java -version
    javac-version
2. Compile o arquivo:
    ```bash
    javac QuebraMaldicao.java
3. Execute:
   ```bash
   java QuebraMaldicao
